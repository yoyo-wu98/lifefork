# LifeFork / 人生岔路

LifeFork 是一个本地优先的自我模拟原型。它通过五个问题和一段可选文本，生成用户的 Self Skill、自我时间线、人生分支路径，并允许用户与不同版本的自己进行对话。它不是命运预测器，而是一面可交互的时间之镜。

## 如何运行

```bash
npm install
npm run dev
```

打开 http://localhost:3000 即可体验。

## V0 功能

- 沉浸式 Landing 页与版本选择（未来/过去/另一条路）。
- 五个核心问题 + 灵感标签输入。
- 可选粘贴材料（localStorage 本地保存）。
- 本地规则生成 Self Skill（价值观/恐惧/模式/原型/证据/声明）。
- 可编辑时间线（修改、删除、反馈）。
- 三条人生路径模拟（A 保持现状、B 激进转向、C 90 天试验）。
- 未来自我聊天（模板回复 + 快捷问题）。
- 危机关键词安全拦截与提示。
- 分享卡片（复制结果、随机箴言、导出 JSON、重新开始）。
- 进度仪式感与徽章反馈。

## 本版本不做什么

- 不做用户注册/登录。
- 不做后端 API。
- 不调用真实大模型。
- 不接入数据库。
- 不做心理诊断、医疗建议或命运预测。

## 数据存储说明

全部数据保存在浏览器 `localStorage`：

- `lifefork.selfSkill`
- `lifefork.currentStep`
- `lifefork.selectedFork`
- `lifefork.chatMessages`

刷新页面后会自动恢复进度。

## 伦理边界

- LifeFork 只做“可能性模拟”，不会给出确定性预测。
- 避免使用“你一定会……”等绝对表达。
- 当检测到危机关键词时，停止模拟并显示现实支持建议。
